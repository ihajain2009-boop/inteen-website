# InTeen - Student Internship Platform

A platform connecting students (9th grade to 2nd year college) with internship opportunities across 50+ categories.

## Features

- **50+ Professional Categories** - Technology, Business, Marketing, Design, Research, Commerce
- **Student-Focused** - Specifically for 9th grade to 2nd year college students
- **Company Portal** - Easy internship listing and management
- **Responsive Design** - Works on all devices

## Pages

1. **Home** (`index.html`) - Landing page with overview
2. **Categories** (`categories.html`) - Browse 50+ internship categories
3. **Company Registration** (`company-registration.html`) - For companies to list internships

## Deployment

This site is deployed on GitHub Pages. The main branch is set to publish from the root directory.

## Local Development

1. Clone the repository
2. Open `index.html` in your browser
3. Or use a local server: `python -m http.server 8000`

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript (ES6+)
- GitHub Pages for hosting
